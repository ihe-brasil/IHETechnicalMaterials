Nesta POC, o aplicativo PacketSender foi utilizado para enviar uma transação ITI-41 para o Syslog Colletor do Gazelle Security Suite.
O PacketSender foi configurado com o Certificado do Gazelle que esta na pasta CA. Os testes foram realizados com a porta TCP e TLS.

Gazelle Security Suite: 
IP/Host: 164.132.1.36
TCP Port 3102
TLS Port 3202
