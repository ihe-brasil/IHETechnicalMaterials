GeneticLoci
Design Comments:

In the context of genetic testing, it is possible to use the interpretationCode to indicate the "overall result" of a panel of loci, e.g., a number of genes, variants, etc. The code attribute could designate the type of relation between the loci, for example: - a genetic test of several genes; - a genetic profile; - a haplotype (e.g., HLA alleles); - a gene expression panel; - several variations along different loci; - genes in a biological pathway; Local codes could be used as well (CWE)
