2.1R_Specimen minimal  (COCT_RM080100UV)

Diagram


Click thumbnail above to open larger graphic in a new window
Description
Parent:   POLB_DM000000UV

The previous version of the R_Specimen minimal CMET had many of the same issues as the previous version of the R_Specimen universal CMET. Since this CMET is a restriction of the universal version, it contains many of the corrections found in the universal version. The minimal version minimzes the information about containers of specimens, focusing mainly on information about the specimen itself.

Information about a specimen, its container or both, includes information that aids in the analysis of a specimen and can be used by automation systems to interpolate settings required for automated analysis (e.g. container type, dilution factor.) Includes aliquot relationships, and container registration events.

Specimen CMET Walk Through

Overview

The Specimen CMET Refined Information Model (RMIM) captures information regarding a specimen and its container. Both the specimen and the source of the specimen are scoped by a specimenEntity which may be located in a container. The specimen itself may be the result of a process and can be acted on by any number of processes in order to handle both the specimen and the container.

The R_Specimen universal CMET is essentially equivalent to the Specimen DMIM. Please see the (The user identified specification HMPOSP is not available) for a description of the Specimen model.
